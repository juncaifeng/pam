# Security Policy

## Supported Versions

We provide security updates for the latest version of Pixel Asset Master.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

If you discover a security issue, report it privately through the maintainer contact channel configured for this repository.

Include in your report:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fix, if you have one

## Scope

This policy covers the Pixel Asset Master source code in this repository, including:

- Python scripts in `skills/pixel-asset-master/scripts/`
- Pixel asset validation and post-processing pipeline
- Project management utilities
- Skill templates and workflow files

Out of scope:

- Third-party AI editors or APIs
- Generated PNG assets, sprite sheets, and export packages
- User-provided reference images or game design documents

## Disclosure Policy

We follow responsible disclosure. Once a fix is available, publish a security advisory or release note if appropriate.
